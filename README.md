# guid-cli

Dead simple CLI-tool to generate a Global Unique Identifiers (GUID, UUID).

## Install

```bash
$ npm install guid-cli -g
```

### Usage

```bash
$ guid
```

Will return

![guid-cli output](/docs/images/output.png)

That's it, dead simple ;-)
